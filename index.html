O que é HTML? Principais comandos

O que é HTML?

HTML é uma das linguagem de marcação mais conhecidas e utilza um formato simples de código que pode ser ferado sem a ajuda de aplicativos especiais.

Para criar um documento HTML, todo o que você precisa é de um editor de texto de ASCII como o NotePade (Bloco de Notas), por exemplo. Se você precisar verificar a formatação do documento durante a criação, pode usar qualquer navegador, pois todos eles são capazes de interpretar o código.

Principais comandos

Os comandos são inseridos no código entre os sinais de menor que "<" e maior que ">" e são chamados de tags.
